## 常用熱鍵

|     指令     |           功能           |
| :--------: | :--------------------: |
|  Ctrl + F9 |           編譯           |
| Ctrl + F10 |       運行上次成功編譯的檔案      |
|     F9     | 編譯並執行檔案（如有 Error 則會報錯） |
| Shift + F2 |       關閉/開啟左側資訊欄       |
