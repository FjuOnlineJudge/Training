    在線評測系統 (Online Judge, OJ)

    在線評量系統提供練習者在寫完題目後，即時上傳並獲得結果。

    • Zerojudge ：中文，有大量簡單題目。
    • TIOJ ：中文，題目偏難。
    • Atcoder ：固定週末晚上有比賽。
    • Atcoder Problem
    • Codeforces ：有比賽但時間不固定。
    • Virtual Judge ：提供提交程式碼到各大 OJ 的平台。
    • Uva ：最古老的 OJ，有 ICPC 題目。
    • Algorithmist
    • Uva toolkit
    • UvaLive ：ICPC 題目。
    • 洛谷
    • neoj ：台大資訊之芽。
