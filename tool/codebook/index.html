    Codebook 準備

    這裡提供常見放到 Codebook 的主題，每支隊伍依據程度選擇放不同的程式碼，程式碼取捨原則如下：

    • 不知道要怎麼用的程式碼優先度最低
    • 難記的程式碼優先度比好記的的程式碼高
    • 常見的程式碼優先度高

    模板

    模板在開局的時候由一位隊員撰寫，之後每題題目先複製一份模板再使用，模板裡會放一些常見的縮寫，通常用 define, using 定義。

    使用 vimrc 可以放入 .vimrc，還可以放編譯的執行的腳本。
